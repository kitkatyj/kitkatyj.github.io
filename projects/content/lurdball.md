# Lurdball

Expanding on my experience of learning to use HTML5 Canvas to create games in school, and building interactive experiences like [[graph-transformer|Graph Transformer]], I wanted to use this experience to recreate a game I played on the hotel TV when on holiday in Kuala Lumpur as a kid.

Unfortunately, I was not able to find any trace of the game online, but I remember it called "Ding Dong", and it involved directing the player towards the goal by manipulating gravity (or it could be just the player can only continuously move in one direction until it hits a wall). IF ANYONE KNOWS WHERE I CAN FIND THAT GAME PLEASE LET ME KNOW!! I do know of this game known as [Tomb Of The Mask](https://en.wikipedia.org/wiki/Tomb_of_the_Mask) that actually works in a very similar way.

But in essence, the game is a technical demo for me to program a game using strictly HTML5 Canvas and Javascript. The goal of the game is to lead a ball towards a goal, but you can only move the ball in 4 cardinal directions (Left, Up, Right, Down - hence LURDball), and the ball does not stop until it hits a wall.

Level data is actually fetched from a /leveldata/ folder in JSON, which defines the walls, the ball, the end, as well as other sprites such as arrows. 

I think there's a lot of potential for this game going forward, and it will probably result in a similar game to Tomb Of The Mask! Perhaps it can be made into 3D too. Who knows?
