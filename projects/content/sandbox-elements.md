# Sandbox Elements

Date: 26 August 2016

[Play now](https://kitkatyj.com/sandbox/)

This is a technical demonstration of using jQuery to create smooth animated user interfaces. 

Inspired by carykh's [Elemental 3](https://youtu.be/J10KDPg_Im0), this interactive demo begins with five elements (Fire, Water, Air, Earth, Energy) in the library, and a template to combine them into many different products.

Each of the five original elements has a corresponding score of 1 for each element. For instance: the original "Water" element has a score of 1 for the Water element, but 0 for the others; The original "Fire" element has a score of 1 for the Fire element; and so on.

When both slots in the combination template are filled, a new product is created and the user is given an opportunity to name this new element. Once confirmed the new element is then added onto the library of elements. This new element's score is the sum of its component elements. For instance: Water and Earth can be combined to a new element which we name "Mud". "Mud" therefore has a Element Score of 1 Water and 1 Earth. The new element also takes the average colours of its component elements and takes an average.

More information about each element's score can be found by right-clicking the element in the library.

While the original Elemental 3 serves as a database for elements created by users, this demo does not provide for that functionality and is outside the scope of the technical demonstration.

Author's Note: Combining two elements together, does not create a new element. Rather it creates a [compound](https://www.bbc.co.uk/bitesize/articles/z8fvsk7#zwbphcw). However the term "element" is used to simplify concepts, and is not intentionally describing compounds as elements.
